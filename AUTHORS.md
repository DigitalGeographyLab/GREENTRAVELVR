## GREENTRAVEL project 

[The GREENTRAVEL project](https://www.helsinki.fi/en/researchgroups/digital-geography-lab/projects/greentravel) (“Greener Urban Travel Environments for Everyone: From measured wellbeing impacts to Big Data analytics”)  is led by Prof. Tuuli Toivonen and runs from 2023–2027, funded by the European Research Council.

## How to cite

If you use this repository in academic or applied work, please cite it as:

> Digital Geography Lab (2025). *GREENTRAVELVR: Source code and documentation for the GREENTRAVEL virtual reality environment*. GitHub repository. Available at: https://github.com/DigitalGeographyLab/GREENTRAVELVR

You may also wish to cite the relevant GREENTRAVEL publications describing the VR cycling experiments and greenery exposure analyses; see the publication list on the GREENTRAVEL project page.
